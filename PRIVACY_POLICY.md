# Privacy Policy for Blob3D

**Last updated:** March 23, 2026

## Overview

Blob3D ("the Game") is a mobile game developed as an independent project. This privacy policy describes how the Game handles user data.

## Data Collection

**Blob3D does not collect, transmit, or share any personal data.**

The Game operates entirely offline and does not communicate with any external servers, APIs, or third-party services.

## Local Storage

The Game uses Unity's PlayerPrefs system to store the following data locally on your device:

- High scores
- Player preferences (e.g., selected skin, sound settings)
- Game progress

This data is stored only on your device and is never transmitted externally. You can delete this data at any time by uninstalling the Game or clearing the app's data through your device settings.

## Analytics and Tracking

The Game does **not** use any analytics services, tracking SDKs, or telemetry of any kind. No usage data is collected or transmitted.

## Advertising

The Game does **not** display advertisements and does not integrate any advertising SDKs.

## Third-Party Services

The Game does **not** integrate with any third-party services, social media platforms, or cloud services.

## Children's Privacy

Since the Game does not collect any personal data, it is compliant with the Children's Online Privacy Protection Act (COPPA) and similar regulations. The Game is safe for users of all ages.

## Changes to This Policy

If this privacy policy is updated, the changes will be reflected in the "Last updated" date above. Users are encouraged to review this policy periodically.

## Contact

If you have any questions about this privacy policy, please open an issue on the project's GitHub repository.
